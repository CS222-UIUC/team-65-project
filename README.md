# team-65-project

### Directory Structure
- 'frontend': React.JS
  - 'Components': Map Components
- 'backend': Flask

### How to Run
- Option 1: Docker
  - Download Docker
  - In Root Directory
    - Start backend and frontend without rebuilding run: docker-compose up
    - If you want to rebuild everything and start run: docker-compose up --build
